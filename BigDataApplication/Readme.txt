--- Приложение для анализа данных на месторождении ---

!!! Для install.sh нужна утилита curl !!!

Измените права на файлы:

	sudo chmod +x install.sh

	sudo chmod +x load_data.sh

	sudo chmod +x analys.sh

Порядок запуска:

	1. install.sh - установка пакетов и запуск Докера с приложением и Базой Данных

	2. load_data.sh - перенос данных из .csv файлов в БД, обработка и загрузка данных Задач в БД.

	3. analys.sh -  построение графиков
